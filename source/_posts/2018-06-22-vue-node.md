终于轮到我自己全权负责一整个后台管理页面，基于前后端分离的基础上，毫不犹豫地选了 vue + element-ui 同学。

# 准备工作

1. 跟紧时代的脚步，使用 [vue-cli v3.0](https://cli.vuejs.org/)

如果你之前安装过 vue-cli ，那你首先要卸载旧版本，如果第一次使用，就直接安装：

```
npm uninstall -g vue-cli
npm install -g @vue/cli
```
